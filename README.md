Convert shared ChatGPT conversations into clean, structured README.md files in seconds.

Chat2Readme CLI automates the process of turning ChatGPT share links into professional documentation. Instead of manually copying, formatting, and organizing content, this tool extracts the conversation and generates a polished README.md ready for GitHub, documentation sites, or project portfolios.

It is especially useful for:

- Developers documenting AI-assisted workflows
- Open-source contributors sharing solutions
- Students creating project documentation
- Engineers converting technical discussions into reusable docs








## Features

- Lightning-fast local search
- Full-text file indexing
- Filename prefix search
- Content search
- Multi-field querying (title + content)
- Relevance-based ranking
- Offline operation
- Lightweight and dependency-friendly
- Extensible architecture
